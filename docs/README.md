# Docs 索引

本目录用于集中管理项目相关的协作、状态与规范文档。**请按下方顺序阅读**。

## 阅读顺序

1. [handoff.md](./handoff.md) — **先读这份**
   - 项目当前状态、背景、待办、风险与已决策事项。
   - 每次开始工作前，先看这里以获取最新上下文。

2. [AGENTS.md](./AGENTS.md) — **再读这份**
   - 执行规范、流程约束、代码 / 测试 / 提交规则。
   - 所有协作行为必须遵守其中的条目。

3. [STRUCTURE.md](./STRUCTURE.md) — **改动代码前必读**
   - 目录结构规范、分层职责、命名与扩展方式。
   - 新增文件请对照此文档，保持结构一致。

## 目录说明

| 文件 | 用途 |
| ---- | ---- |
| `handoff.md` | 项目状态、背景、待办、风险、已决策事项 |
| `AGENTS.md` | 项目协作规则与执行规范 |
| `STRUCTURE.md` | 目录结构规范与扩展约定 |
| `README.md` | 本文档，说明先读哪个 |
